## Kurallar

1. Format: `"Söz metni." — Yazar Adı`
2. Yazar adını **değiştirme** (Latin harflerle kalır: Muhammad Ali, Pelé, Naim Süleymanoğlu…)
3. Söz gövdesi **en fazla ~15 kelime**, fitness/antrenman/azim tonu
4. Spor dalı adı kullanma (futbol, boks, maraton, kort, saha vb.)
5. **107 satır**, sıra numarası aynı kalmalı (#1 → #1)
6. Bana JSON dosyası olarak gönder: `de.json`, `fr.json` vb.

## Bana nasıl göndereceksin?

**Seçenek A — JSON dosyası (tercih edilen)**

Her dil için bir dosya. Örnek `de.json`:

```json
[
  "\"Zähle nicht die Tage, sondern lass die Tage zählen.\" — Muhammad Ali",
  "\"Ich hasste jede Minute des Trainings, aber ich sagte: Gib nicht auf.\" — Muhammad Ali",
  "..."
]
```

Dosyaları `lib/motivation-quotes/incoming/` klasörüne koy veya chat'e yapıştır.

**Seçenek B — Chat'te dil dil**

Mesajında şunu yaz:
```
DIL: de
1. "..." — Muhammad Ali
2. "..." — Muhammad Ali
...
107. "..." — Nurcan Taylan
```
